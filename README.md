# 📊 Samsung Global Sales Performance Dashboard (Power BI)

---

## 📌 Project Overview
This project presents an *interactive Power BI dashboard* analyzing the global sales performance of Samsung products. 
The dashboard provides insights into *revenue, units sold, customer ratings, sales channels, regions, and product categories*.

It helps stakeholders understand:
- Global sales distribution
- Customer behavior
- Product performance
- Sales channel contribution
- Customer rating trends
